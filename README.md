### Go
*   Домашние задания 1 семестра являются реализации различных алгоритмов и структур, направленые на изучения языка (основы, работа с ошибками, многопоточное программирование и др.)
*   Домашние задания 2 семестра будут направлены на реальную разработку и знакомство с различными инструментами, итогом должен стать готовый pet-проект
